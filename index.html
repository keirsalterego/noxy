Welcome to Stratum

Stratum sits between your apps and the internet and gives you real control over how packets leave. Private DNS over HTTPS, pluggable routing through direct, Tor, I2P, or chains, and certificate pinning that fails loud. One small binary, no cloud required. // What It Does
ResolutionPrivate DNS

Names resolved over DNS-over-HTTPS so the network can't read every lookup. TTL cache built in.

 RoutingPluggable Transports

Direct, Tor, I2P, or chained. Tor and I2P are just SOCKS upstreams on a port; the core never special-cases them.
ControlRule-Based Egress

Send *.onion to Tor, *.i2p to I2P, the rest where you choose. First match wins.

 IntegrityCertificate Pinning

SPKI-SHA256 pins on the upstreams it terminates. A swapped key fails the connection and raises an alert.
OperationsLive Dashboard

A terminal cockpit for connections, routes, and config reloads, all without a restart. Or run headless.

 FoundationBuilt Carefully

Non-blocking epoll loop, bounds-checked parsers, capped buffers. Tested under ASan, UBSan, and valgrind.
